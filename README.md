Getting Started
Prerequisites
Eclipse IDE
Git


Importing the Project into Eclipse
Clone the repository:

Open a terminal and run the following command to clone the repository:

bash
Copy code
git clone https://github.com/Elkepun/StaffManagementSystem.git
cd StaffManagementSystem
Import the project into Eclipse:

Open Eclipse IDE.
Go to File > Import.
Select Existing Projects into Workspace under the General category and click Next.
Click Browse and select the directory where you cloned the repository.
Ensure the project is selected in the Projects list and click Finish.
